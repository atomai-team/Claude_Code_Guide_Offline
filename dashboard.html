把 Claude Code 用顺手的四件事 · 键盘快捷键 · 启动参数 · 权限模式 · 配置升级

⌨️ 键盘快捷键 · 必背 10 个

手指不离键盘，效率翻倍

CtrlC取消当前操作它跑超时或做错了，立刻停
CtrlR反向搜索历史命令想找上次用过的 prompt
CtrlL清屏（保留历史）屏幕太乱，快速整理
↑↓翻历史 prompt重复上次类似的请求
EscEsc中断当前任务它跑偏了，立刻打断
ShiftTab循环切会话模式默认 → acceptEdits → planMode
!在会话内跑 shell!git status 输出进对话可分析
@直接喂文件给 Claude看 @src/app.js 有什么 bug
/触发补全（斜杠命令 / 名字）输入框打 / 显示所有命令
CtrlD退出当前会话结束当天工作（CLI）

Shift + Tab 三模式循环 — 默认（每步确认，最安全）→ acceptEdits（文件编辑自动，最快）→ planMode（只读分析，最稳）。

🛠️ CLI 启动参数

在终端这一层就定好

--print · -p非交互模式，输出完即退
--continue · -c继续最近一次会话
--resume列出 / 恢复历史会话
--model NAME指定模型（opus / sonnet / haiku）
--add-dir PATH把额外目录加进上下文
--permission-mode MODEdefault / acceptEdits / plan / bypassPermissions
--append-system-prompt追加系统提示词
--dangerously-skip-permissionsYOLO 模式（慎用）
--verbose详细日志输出
--help看所有 CLI 参数

例 — claude --model sonnet --add-dir ../shared · cat error.log | claude -p "分析日志"

🌐 环境变量

高级配置与开关

ANTHROPIC_API_KEY官方 API Key（必填 · 禁代理直连）
ANTHROPIC_BASE_URL自定义端点（默认 api.anthropic.com）
ANTHROPIC_MODEL默认模型（覆盖 settings.json）
ANTHROPIC_DEFAULT_OPUS_MODELopus 路由别名解析
CLAUDE_CODE_ENTRYPOINT入口标识（cli / sdk）
DISABLE_OMC关闭 OMC 全部 hook（kill switch）
OMC_SKIP_HOOKS跳过指定 hook（逗号分隔）
OMC_STATE_DIR改 state 根目录（默认 ./.omc/）
DISABLE_TELEMETRY关闭匿名使用统计

🔒 配置防改铁律 — settings.json 的 model + enabledPlugins + env.ANTHROPIC_* 三项 AI 不得擅改，豁免仅限你 /model 或对话明示。

🔐 权限模式 · 4 种

放多少权限给它，你说了算

default每步要按 Y 确认正式项目 · 最稳妥
acceptEdits文件编辑自动，命令仍确认迭代开发 · 最常用
plan只读不写，出方案等你批大改之前先理解
bypassPermissions全部允许，YOLO 全自动⚠️ 慎用 · 仅限可逆短任务

切换方式 — 会话里按 Shift + Tab 循环，或启动时加 --permission-mode MODE。
